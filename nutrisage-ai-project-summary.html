🏗 Architecture

Backend (app.py)

  • Flask 3 with flask-cors
  • ibm-watsonx-ai SDK (chat API)
  • Granite 3.3 8B Instruct model
  • Mifflin–St Jeor BMR/TDEE formula
  • Watsonx singleton (lazy init)
  • Demo mode fallback (no key needed)
  • SYSTEM_PROMPT injected per request

Frontend

  • Bootstrap 5.3 + Bootstrap Icons
  • 6 tabs: Chat, Dashboard, Meal, BMI, Family, Analyzer
  • Dark/light mode + localStorage persist
  • Animated messages + typing indicator
  • Real-time char counter + Ctrl+Enter
  • Macro breakdown bars (animated)
  • Mobile offcanvas + bottom tab bar

Security

  • All secrets in .env (never in code)
  • python-dotenv for safe loading
  • Safety rules baked into SYSTEM_PROMPT
  • FLASK_DEBUG=False for production
  • .env.example (no real keys committed)

Deployment Options

  • Local: python app.py
  • Prod: gunicorn -w 2 app:app
  • Docker: Dockerfile included in README
  • IBM Code Engine (serverless)
  • Railway / Render / Fly.io
